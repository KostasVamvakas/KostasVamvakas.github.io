PDB — Control

Power distribution and structured connectivity for control electronics.

← Back to PCB list

Provides organized power distribution and connector structure for the control stack, improving maintainability and integration.

Role
Schematic capture + PCB layout
Tools
KiCad · DFM-minded layout
Context
Active control of the foils required a compact controller capable of real-time control, sensor interfacing, and reliable actuation. A dedicated board was needed to tightly integrate the microcontroller with the required I/O.
Outcome
The board enabled precise and responsive foil control, with stable operation during testing. The integration simplified wiring and improved maintainability.

Design highlights

  • Teensy-based control architecture.
  • Interfaces for sensors and actuators.
  • Designed for deterministic, real-time control applications.
